My REST API

This API manages Users, Income, and Expenses. It helps you store and manage financial data in Firebase.

Available Endpoints

Users

  • GET /users
    Get a list of all users from the database
  • POST /users
    Add a new user to the database
  • PUT /users/:id
    Update user details using ID
  • DELETE /users/:id
    Delete a user using ID

Expenses

  • GET /expenses
    Get all expenses from the database
  • POST /expenses
    Add a new expense
  • PUT /expenses/:id
    Update an expense using ID
  • DELETE /expenses/:id
    Delete an expense using ID

Income

  • GET /income
    Get all income records
  • POST /income
    Add new income data
  • PUT /income/:id
    Update income using ID
  • DELETE /income/:id
    Delete income using ID

Click the GET links to see data from Firebase. Use Postman or Thunder Client for POST, PUT, and DELETE.
